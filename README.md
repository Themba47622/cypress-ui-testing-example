# UI Test Automation | Cypress

## Description
cypress ui test automation example

## Installation
```bash
$ npm install
```

## Run tests
```bash
$ npm test
```

## Run tests with mochawesome reporter
```bash
$ npm test --reporter mochawesome
```

## Open Cypress
```
$ npm run cy:open
```
